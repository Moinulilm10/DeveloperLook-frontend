# DeveloperLook Frontend Assessment

Welcome to my submission for the Junior MERN Developer frontend assignment at DeveloperLook!

This project is a high-fidelity, pixel-perfect recreation of the [GetHyped](https://www.gethyped.nl/) homepage, strictly following the provided assignment parameters.

**🚀 Live Demo:** [https://developer-look-frontend.vercel.app/](https://developer-look-frontend.vercel.app/)

---

## 🎯 Task Scope & Objectives

The goal of this assignment was accurately recreating the target website focusing purely on frontend presentation and interactive fidelity:

- ✔️ **The Homepage:** Fully constructed with precision to match structure, component scaling, and layout alignments.
- ✔️ **Mobile Version:** Built fully responsive using mobile-first tailwind methodologies. Transitions seamlessly between Desktop, Tablet, and Mobile breakpoints.
- ✔️ **Hamburger Menu:** Includes exact implementations of the mobile navigation transitions and behaviors.
- ✔️ **Interactive Physics:** Fanned deck animations ("Parting the Sea" logic) and hovering states were identically replicated using Framer Motion exactly as seen on the live site.

## 🛠 Tech Stack

- **React / Vite** - Fast local development environment and optimized production builds.
- **Tailwind CSS v4** - Used for comprehensive, utility-first styling ensuring pixel-perfect spacing and typography matching.
- **Framer Motion** - Drives the advanced physics components, interactive deck splitting algorithms, and smooth hover behaviors natively.

## 💻 Local Setup Instructions

To run this project locally on your machine, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Moinulilm10/DeveloperLook-frontend.git
   cd DeveloperLook-frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Spin up the development server:**
   ```bash
   npm run dev
   ```

4. **View the Application:**
   Open your browser and navigate to `http://localhost:5173` (or the port specified in your terminal).
